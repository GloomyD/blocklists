### What is this?
Some websites spread misleading content, low-quality AI-generated content ("AI Slop"), or fake news, sometimes with political intent.
This repository provides curated filter lists for **uBlacklist** and **NextDNS** to help you block these unwanted contents.

### Supported Tools
- **uBlacklist**: Browser extension (Chrome, Firefox, Safari) that filters Google Search results.
  [Official website](https://ublacklist.github.io/)
- **NextDNS**: DNS resolver that blocks trackers, ads, and unwanted content at the network level.
  [Official website](https://nextdns.io/?from=zjmur6v8)

### How to Use
You can:
- Manually add sites to block
- Subscribe to public filter lists
- Copy-paste lists from this repository
